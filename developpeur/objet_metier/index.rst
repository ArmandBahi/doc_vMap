Objets métiers
====================================

Les objets métiers dans Vmap permettent l'interrogation et l'édition des objets stockés en base de données. Une fois déclaré, un objet métier est construit dans le studio dans lequel sont spécifiés chacun des formulaires d'affichage, de recherche, d'insertion et d'édition.
